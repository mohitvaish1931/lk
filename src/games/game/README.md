# 3D History Game - Interactive Educational Experience

An immersive 3D educational game built with React Three.js for students of classes 6 to 8, based on NCERT History curriculum.

## üéÆ Game Features

### Core Gameplay

- **Futuristic Time Machine**: 3D cockpit with student character (12-14 years old)
- **Interactive Time Travel**: Dynamic animations with light tunnels and spinning clocks
- **Historical Settings**: 5 different historical periods from NCERT curriculum
- **Educational Quizzes**: MCQ questions with immediate feedback
- **Kid-Friendly UI**: Colorful, responsive design for tablets and laptops

### Historical Periods Covered

1. **Harappan Civilization** (2600-1900 BCE)

   - Indus Valley civilization questions
   - Agriculture and river-based questions

2. **Mauryan Empire** (322-185 BCE)

   - Chandragupta Maurya and Ashoka
   - Pataliputra and Buddhist influence

3. **Delhi Sultanate** (1206-1526 CE)

   - Qutb-ud-din Aibak and Qutub Minar
   - Administrative systems

4. **Mughal Empire** (1526-1857 CE)

   - Babur, Akbar, and Shah Jahan
   - Taj Mahal and cultural achievements

5. **Gupta Empire** (320-550 CE)
   - Golden Age of Ancient India
   - Chandragupta I and Kalidasa

### Technical Features

- **Three.js 3D Rendering**: Immersive 3D environments
- **React Three Fiber**: Modern React integration
- **Framer Motion**: Smooth animations and transitions
- **Tailwind CSS**: Responsive, modern styling
- **React Router**: Navigation between game states
- **Lucide React Icons**: Beautiful, consistent iconography

## üéØ Game Flow

1. **Cockpit Scene**: Student sits in futuristic time machine
2. **Time Travel**: Click "Move to Past" to start journey
3. **Historical Scene**: Arrive at random historical period
4. **Quiz Time**: Answer MCQ questions about the period
5. **Feedback**: Get immediate feedback with explanations
6. **Return**: Go back to time machine for next adventure
